# Malik Hashir — Cyberpunk Portfolio

A futuristic, production-ready cyberpunk-themed personal portfolio built with Next.js 14, Tailwind CSS, Framer Motion, and Three.js.

## Tech Stack
* **Framework:** Next.js 14 (App Router) + TypeScript
* **Styling:** Tailwind CSS + Custom CSS Variables (Glassmorphism, Neon Glow)
* **Animations:** Framer Motion
* **3D Models:** Three.js + @react-three/fiber + @react-three/drei
* **Particles:** Canvas-based Neural Network animation
* **Icons:** Lucide React + React Icons
* **Contact Form:** EmailJS

## How to Run

1. **Verify Prerequisites:** Make sure you have Node.js 18.17+ installed.
2. **Navigate to the project folder:**
   ```bash
   cd malik-portfolio
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Environment Setup (Optional):**
   To make the contact form work, update the EmailJS credentials in `lib/data.ts`.
5. **Run the development server:**
   ```bash
   npm run dev
   ```
6. **Open in browser:**
   Visit: http://localhost:3000
